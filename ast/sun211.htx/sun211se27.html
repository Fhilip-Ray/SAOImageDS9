E UNIX Command Descriptions

The commands described here are provided for use from the UNIX shell to assist with developing software which uses AST. To use these commands, you should ensure that the directory “/star/bin”39 is on your PATH.

39Or the equivalent directory if AST is installed in a non-standard location.
