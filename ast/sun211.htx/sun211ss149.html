astPurgeWCS

Delete all cards in the FitsChan describing WCS information

Description:

This function deletes all cards in a FitsChan that relate to any of the recognised WCS encodings. On exit, the current card is the first remaining card in the FitsChan.

Synopsis

void astPurgeWCS( AstFitsChan ∗this )

Parameters:

this
Pointer to the FitsChan.
