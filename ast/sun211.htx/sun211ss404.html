SpecVal

The spectral position at which flux values are measured

Description:

This attribute specifies the spectral position (frequency, wavelength, etc.), at which the values described by the FluxFrame are measured. It is used when determining the Mapping between between FluxFrames.

The default value and spectral system used for this attribute are both specified when the FluxFrame is created.

Type:
Floating point.

Applicability

FluxFrame
All FluxFrames have this attribute.
