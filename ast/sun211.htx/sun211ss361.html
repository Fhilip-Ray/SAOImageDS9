Nrow

The number of rows in the table

Description:

This attribute holds the index of the last row to which any contents have been added using any of the astMapPut... AST_MAPPUT... functions. The first row has index 1.
Type:
Integer, read-only.

Applicability

Table
All Tables have this attribute.
