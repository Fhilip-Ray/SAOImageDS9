Ncard

Number of FITS header cards in a FitsChan

Description:

This attribute gives the total number of FITS header cards stored in a FitsChan. It is updated as cards are added or deleted.
Type:
Integer, read-only.

Applicability

FitsChan
All FitsChans have this attribute.
