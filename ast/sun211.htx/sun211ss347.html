Naxes

Number of Frame axes

Description:

This is a read-only attribute giving the number of axes in a Frame (i.e. the number of dimensions of the coordinate space which the Frame describes). This value is determined when the Frame is created.
Type:
Integer, read-only.

Applicability

Frame
All Frames have this attribute.
FrameSet
The Naxes attribute of a FrameSet is the same as that of its current Frame (as specified by the Current attribute).
CmpFrame
The Naxes attribute of a CmpFrame is equal to the sum of the Naxes values of its two component Frames.
