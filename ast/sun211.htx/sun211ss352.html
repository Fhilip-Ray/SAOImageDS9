Nframe

Number of Frames in a FrameSet

Description:

This attrbute gives the number of Frames in a FrameSet. This value will change as Frames are added or removed, but will always be at least one.
Type:
Integer, read-only.

Applicability

FrameSet
All FrameSets have this attribute.
