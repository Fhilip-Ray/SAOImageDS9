Nparameter

The number of global parameters in the table

Description:

This attribute holds the number of global parameters currently in the table. Parameters are added and removed using the astAddParameter and astRemoveParameter functions.
Type:
Integer, read-only.

Applicability

Table
All Tables have this attribute.
