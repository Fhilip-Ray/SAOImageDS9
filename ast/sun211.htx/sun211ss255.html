Bounded

Is the Region bounded?

Description:

This is a read-only attribute indicating if the Region is bounded. A Region is bounded if it is contained entirely within some finite-size bounding box.
Type:
Integer (boolean), read-only.

Applicability

Region
All Regions have this attribute.
