CardType

The data type of the current card in a FitsChan

Description:

This attribute gives the data type of the keyword value for the current card of the FitsChan. It will be one of the following integer constants: AST__NOTYPE, AST__COMMENT, AST__INT, AST__FLOAT, AST__STRING, AST__COMPLEXF, AST__COMPLEXI, AST__LOGICAL, AST__CONTINUE, AST__UNDEF.
Type:
Integer, read-only.

Applicability

FitsChan
All FitsChans have this attribute.
