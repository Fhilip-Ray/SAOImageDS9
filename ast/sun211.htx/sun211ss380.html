RegionClass

The AST class name of the Region encapsulated within an Stc

Description:

This is a read-only attribute giving the AST class name of the Region encapsulated within an Stc (that is, the class of the Region which was supplied when the Stc was created).
Type:
String, read-only.

Applicability

Stc
All Stc objects this attribute.
